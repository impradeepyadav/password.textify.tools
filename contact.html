Before You Email: Quick Answers

We love helping users, but to save you time, please check these common topics first:

  • Can you recover a password I lost? No. As stated on the homepage, we do not store or transmit any passwords. We have no database to recover from. Please use a password manager.
  • Is this website safe? Yes. You can verify this by disconnecting your internet after the page loads—it will still generate passwords. That's proof it runs entirely on your device.
  • Do you have an API? No. We do not offer an API because sending passwords over the internet defeats the purpose of client-side, zero-knowledge security.

Frequently Asked Support Questions

The generator isn't working in my browser. What should I do?
Try updating your browser to the latest version. We officially support Chrome, Firefox, Safari, and Edge. If the problem persists, clear your browser cache or try incognito mode. If you're using an older browser like Internet Explorer, the Web Crypto API may not be supported.
Are the passwords really random and secure?
Yes. We use the Web Crypto API, which provides cryptographically secure random numbers. This is the same standard used by banks and governments for secure encryption. You can view the page source or run a network monitor to confirm no data leaves your browser.
Can I use Password Generator offline?
Absolutely. Once the page loads, you can disconnect from the internet and it still works perfectly. All generation happens locally in your browser's memory.
I lost a password I generated. Can you recover it?
No, we cannot recover any passwords. Since we operate on a zero-knowledge architecture and don't store anything, we have no way to retrieve passwords. Please save your passwords in a password manager immediately after generation.
How can I support the project?
Thank you for asking! The best way to support us is to share the website with friends, family, or on social media. Spreading awareness about secure, client-side password generation helps make the internet a safer place for everyone. We do not accept donations at this time.
